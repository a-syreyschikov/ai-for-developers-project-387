# Backend

Backend реализован на .NET 8 + ASP.NET Core Minimal API и следует контракту из `contracts/api/main.tsp`.

## Запуск через Docker

```bash
make backend-build
make backend-run
```

API будет доступен на `http://localhost:8080`.

## Тесты через Docker

```bash
make backend-test
```

Локальная установка `dotnet` не требуется.

## Данные

Хранилище работает в памяти. После перезапуска сервиса созданные типы событий, расписание и бронирования сбрасываются.

Начальные тестовые данные на русском языке:

- владелец: `Алексей Петров`;
- тип события: `Вводная встреча`, 30 минут;
- расписание: понедельник-пятница, `09:00-18:00`, часовой пояс `Europe/Moscow`.
